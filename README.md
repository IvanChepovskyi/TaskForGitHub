# TaskForGitHub

Установите и настройте Git на своей домашней машине. Создайте репозито-
рий, в котором под управление Git проделайте следующие действия:

5a. Напишите на Octave скрипт, реализующий метод деления отрезка попо-
лам для нахождения точки минимума функции одной переменной. При
этом постарайтесь за время написания кода сделать несколько комми-
тов, не скупясь на поясняющий комментарий к нему. Просматривайте,
как изменяется «история кода» при помощи утилиты gitk.

5b. Создайте ветку, в которой добавьте новую функциональность к вашему
коду — реализуйте отрисовку графика минимизируемой функции и то-
чек на оси абсцисс, являющихся последовательными приближениями в
методе деления пополам. Не забывайте коммитить!

5c. Вернитесь к главной ветке и выполните слияние. Зафиксируйте состо-
яние репозитория, запакуйте его в архив .zip (обязательно zip !) с на-
званием вида Фамилия_Имя_ДЗ1.zip (кириллицей). Отправьте его на
адрес ogulenko.a.p@onu.edu.ua .
